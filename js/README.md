# bilibili-share-js

一个篡改猴插件，用于替代B站网页端自带的分享功能，可以复制出标题、作者、短链和视频封面。

## 功能特性

- 替换B站视频页面的原生分享按钮，点击后自动复制富文本信息
- 支持通过油猴菜单命令手动触发分享
- 提取视频的 BV 号、标题、作者、封面图、短链接
- 将信息格式化为 HTML 并复制到剪贴板，可直接粘贴到支持富文本的编辑器（如 Word、QQ、微信等）
- 自动监听动态加载的分享按钮，适配单页应用

## 安装方法

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 或类似用户脚本管理器
2. 点击 [安装链接](https://github.com/Liang457/bilibili-share-js/raw/refs/heads/main/src/main.js)，按照提示安装脚本

## 使用方法

### 自动触发
访问任意 B 站视频页面（如 `https://www.bilibili.com/video/BV1xx...`），点击页面上的分享按钮（通常在视频工具栏左侧），脚本会自动复制视频信息。

### 手动触发
点击浏览器工具栏中的 Tampermonkey 图标，选择菜单中的「📤 富文本分享视频」，同样会执行复制操作。

### 复制内容示例
复制后的 HTML 格式如下（可直接粘贴到支持富文本的编辑器中）：

```
「视频标题」——作者名<br><a href="https://b23.tv/BV1xx...">https://b23.tv/BV1xx...</a><br><img src="封面图地址" alt="视频标题">
```

## 注意事项

- 脚本使用了 B 站的公开 API，首次调用时可能会提示「脚本试图访问跨域资源」，请勾选「始终允许」避免重复提示
- 仅在 `https://www.bilibili.com/video/*` 页面生效
- 若页面未检测到 BV 号或 API 请求失败，会通过通知提示错误信息
